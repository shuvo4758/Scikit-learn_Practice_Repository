# Scikit-learn_Practice_Repository
Machine Learning Practice – Regression &amp; Evaluation Metrics


This repository contains Jupyter notebooks created while learning **basic machine learning concepts using Python and scikit-learn**.  
The focus is on **regression models** and **model evaluation metrics** with simple, easy-to-understand examples.

---

## 📂 Files in This Repository

### 1️⃣ `Regression.ipynb`
This notebook covers:
- Introduction to regression concepts
- Using **scikit-learn** models
- Working with sample datasets
- A simple **K-Nearest Neighbors (KNN)** classification example
- Predicting outcomes based on user input (e.g., Apple vs Orange classification)

**Concepts used:**
- Linear Regression
- KNN Classifier
- Model training and prediction
- Feature input handling

---

### 2️⃣ `MAE_MSE_RMSE.ipynb`
This notebook focuses on **model evaluation metrics**, including:
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

**Concepts used:**
- Evaluating model performance
- Understanding prediction errors
- Comparing actual vs predicted values
